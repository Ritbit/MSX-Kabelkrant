# Contributing

This repository preserves a historical MSX2 BASIC project.

Contributions are welcome, but the main goal is preservation rather than modernization.

## Guidelines

- Preserve the original behaviour where possible.
- Do not rewrite code just to make it look modern.
- Keep historical source files intact.
- Put cleaned or converted variants in clearly named folders.
- Document any change that affects runtime behaviour.
- Prefer small, reviewable changes.
- Test changes in an MSX2 emulator when possible.

## Useful contributions

Examples of useful contributions:

- emulator setup notes
- screenshots or videos
- documentation improvements
- file format documentation
- corrections to historical notes
- BASIC listing cleanup that preserves behaviour
- verified notes about MSX2 compatibility

## Historical accuracy

If you are unsure whether a change is historically accurate, document it as an interpretation or reconstruction rather than as fact.
