# Module Reference

This document briefly describes the major source modules.

| Module | Purpose |
|--------|---------|
| AUTOEXEC.BAS | Boot entry point |
| KBLINIT.SYS | Initialises the application |
| LOOP.SYS | Main display loop |
| MAIN.SYS | Main operator interface |
| SYSTEM.SYS | Shared system routines |
| UTILS.SYS | Utility routines |
| TEKST.SYS | Text handling/editor support |
| KRANT.SYS | Newspaper/page management |
| RAMDISK.BIN | Memory-backed storage support |

The exact responsibilities of each module should be refined as the code is documented.
