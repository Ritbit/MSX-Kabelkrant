# Data Files

## Graphics

### KRANT4.SC7

SCREEN 7 graphics used by the application. This includes graphical assets and the proportional font used by the renderer.

## Font metrics

### X.DAT

Stores glyph width or X-coordinate information required by the proportional renderer.

### XK.DAT

Additional font metadata used together with X.DAT.

## Page data

The application loads page definitions and textual content from disk before copying frequently accessed information into the RAM disk.

The exact on-disk format will be documented in a future revision.
