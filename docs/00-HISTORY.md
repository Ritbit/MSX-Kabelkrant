# History

Kabelkrant-MSX2 is the preserved source of a hospital information/bulletin system written for MSX2.

The published code represents a later sixth-edition generation of the project. Earlier editions existed before this codebase, including at least a possible 5.4 generation that may be archived separately later.

The system ran on a Philips NMS-8250 connected to the local hospital television distribution system. Its purpose was practical: display menus, visiting hours, announcements and local hospital information continuously for patients, visitors and staff.

This repository documents the original BASIC implementation. It intentionally avoids modernising the design: the value is in preserving how the system actually worked on MSX2-era hardware.
